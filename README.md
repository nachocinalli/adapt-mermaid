# adapt-mermaid

**adapt-mermaid** is an extension for the Adapt Framework that allows you to add [mermaid](http://mermaid.js.org/) diagrams to your content.
You can see it [here](https://adaptlearning-no-core.web.app/#/id/expo-55).
